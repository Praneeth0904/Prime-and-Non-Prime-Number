## 📌 Prime and Non-Prime Number

Validates whether the number is prime or non-prime from user's input.

---

## 🛠️ Tech Stack

HTML5 • CSS3 • JavaScript

---

## ✨ Features

* Checks the number if it is a prime or a non-prime number.
* Engaging UI which displays green for prime and red for non-prime.
